* org-history
*Version: 0.1*

Show the last modification date for headers in Org-mode of GNU Emacs using git VCS and auto-commit.

* Features
- No buttons: simply enable the org-history-mode minor mode in the Org buffer.
- Auto-commit and git initialization are requested once from the user.
- Accurate tracking of the current file is enabled via the .dir-locals.el file in the current directory.

* Configuration

#+begin_src elisp :lexical t :results none :exports code :eval no
(add-to-list 'load-path "/path-to/emacs-org-history")
(require 'org-history)
#+end_src

Activation: ~M-x org-history-mode~

* Customization

: M-x customize-group RET org-history
